# easydb-export-transport-ftp-plugin

Export plugin for **easydb5** and **fylr** which adds transports to FTP and WEBDAV.

## Installation for fylr

To install this plugin in fylr, open the Plugin Manager and add a new plugin with type "url". 

This URL always links to the latest released version of the plugin: 

```
http://github.com/programmfabrik/easydb-export-transport-ftp-plugin/releases/latest/download/easydb-export-transport-ftp-plugin.zip
```
